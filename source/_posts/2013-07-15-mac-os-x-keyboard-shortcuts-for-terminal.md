Mac OS X 的 terminal 是 BASH, 以下是 BASH 的快捷鍵:


	Ctrl + A    回行首
	Ctrl + E    去行尾
	Ctrl + L    清劃面
	Ctrl + U    清掉游標前的文字
	Ctrl + H    刪一個字元
	Ctrl + R    找執行過的指令 (好用)
	Ctrl + C    中斷執行
	Ctrl + D    跳出目前的shell
	Ctrl + Z    把目前執行的指令設到背影.要回到前景可透過 fg 指令
	Ctrl + W    刪除游標之前的字
	Ctrl + K    清除游標之後所有內容
	Ctrl + T    交換字元
	Esc + T  	交換"字"
	Alt + F  	往前移一個字
	Alt + B  	往後移一個字
	Tab      	自動完成
